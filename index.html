HTML

  • The head element contains information, or metadata, that helps the browser to render the page correctly.
  • Elements within the haed are not visible when a user looks at a webpage.
  • The head element defines 3 elements:
  • 1.charset:sets character encoding
  • 2.http-equiv:sets a HTTP header
  • 3.viewport:sets default view for our webpage
  • The body element represents the visible content shown to the user.
  • All the elements we want the user to see and interact with are placed inside the body element.
  • If there are 3 elements nested inside of the body element, those 3 nested elements are known as the child elements
  • The element that holds all the childern is known as the parent element
  • This parent/child language is often used by developers to explain the relationship among elements in HTML.
  • Metadata is data about our webpage that the browser uses to effectively serve our webpage users.
  • Metadata is typically not seen by users.
